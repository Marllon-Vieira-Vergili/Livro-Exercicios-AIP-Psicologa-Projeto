 <h1>Livro de Exercícios AIP - Projeto para Psicóloga de Carreira</h1>

Plataforma digital para exercícios terapêuticos desenvolvida para psicóloga autônoma. baseado no livro de exercício AIP, de exclusividade da editora Vetor, conforme o modelo escaneado no link abaixo.

⚠️ <mark>ATENÇÃO:</mark> Este projeto teve que ser pausado por tempo indefinido, pois a cliente está tentando buscar autorização da editora para trazer esse formulário exclusivo digitalmente. <span>Não me responsabilizo por copyrights ilegais.</span>

<h2>Link: <span>https://pt.scribd.com/document/604444399/Livro-de-Exercicio-AIP</span></h2>

## Tecnologias utilizadas

- React
- Vite
- TypeScript
- CSS

## Como rodar

```bash
git clone https://github.com/Marllon-Vieira-Vergili/Livro-Exercicios-AIP-Psicologa-Projeto.git
cd Livro-Exercicios-AIP-Psicologa-Projeto
npm install
npm run dev
Acesse: http://localhost:5173
```
